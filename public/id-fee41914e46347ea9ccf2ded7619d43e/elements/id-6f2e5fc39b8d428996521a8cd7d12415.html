Verkehrsverbände ()
Stakeholder
Anforderungen
Verkehrsverbände Das System muss Echtzeitdaten über Verkehrsinformationen mit einer Aktualisierungsfrequenz von maximal 10 Sekunden bereitstellen, damit Fahrgäste aktuell informiert werden können.
Verkehrsverbände Das System muss öffentliche Verkehrsdaten aus mindestens fünf verschiedenen Quellen integrieren, damit eine umfassende Vernetzung ermöglicht wird.
